# 數獨
目前提供 ```9 * 9``` 及 ```16 * 16``` 大小的數獨方格，當所有格子都被填滿則結束遊戲。<br />
Release: [Sudoku-v11.17.24](https://github.com/zylin1998/Sudoku/releases/tag/Sudoku-11.17.24)
## 規則
所有橫排、直排及去區域都不能出現相同數字。
## 功能
1. 可以選擇以 ```3``` 或 ```4``` 危機底進行遊戲。
2. 可以選擇初始顯示的數字數量。
3. 可以透過檢視按鈕檢視場上特定數字的位置
4. 提供少量提示次數方便解題
5. 可直接顯示答案。
